# Sobre a Aventura

Projetada para **1 a 3 jogadores** com personagens de **1º nível** ou superior.

| Campo | Valor |
|---|---|
| Duração estimada | 2 horas |
| Sessões sugeridas | 1 sessão de ~2h |
| Nível de início | 1 |
| Nível de fim | 2 |
| Jogadores | 1–3 |
| Tom | Fantasia Sombria |

Os aventureiros explorarão as profundezas gélidas de uma cripta sob um antigo templo outrora pacífico, enfrentando a névoa e os mortos que se ergueram de seus túmulos.
