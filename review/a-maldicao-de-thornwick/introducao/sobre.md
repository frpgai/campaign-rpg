# Sobre a Aventura

Projetada para **1 a 4 jogadores** com personagens de **1º nível**.

| Campo | Valor |
|---|---|
| Duração estimada | ~2 horas |
| Sessões sugeridas | 1 sessão de ~2h |
| Nível de início | 1 |
| Nível de fim | 2 |
| Jogadores | 1–4 |
| Tom | Sombrio / Horror Investigativo |

Os jogadores chegam a uma aldeia sitiada pelo próprio passado: mortos que caminham, habitantes que não dormem e um segredo de três décadas enterrado junto com uma injustiça que ninguém admitiu. Para resolver o problema, não bastará combater — será preciso compreender.
